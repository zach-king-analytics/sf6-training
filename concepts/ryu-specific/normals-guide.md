# Ryu Normals Guide

**Definition:** The set of Ryu's core normal attacks and their optimal range, role, and risk profile — establishing which button to press in which situation and what can go wrong with each.

## Core Normals

| Normal | Role | Range | Cancel Route | Risk |
|--------|------|-------|-------------|------|
| st.MP | Pressure check / confirm | Close-mid | DR cancel on hit | Whiffed at bad range |
| cr.MK | Poke / cancel starter | Mid | DR cancel when stable | Counter-hit if predictable |
| st.HK | Counter poke / whiff punish | Mid | None (standalone) | Large whiff window; avoid autopilot |
| cr.HP | Anti-air / close challenge | Close | Combo on hit | Spacing sensitive; loses to crossup |
| cr.LK | Low poke / frame trap | Close | cr.LP link | Beaten at mid range |
| st.HK (far) | Space control | Full mid | None | Slow; do not use as a check |

## Failure Modes

| Failure | Cause | Correction |
|---------|-------|------------|
| Poke rhythm becomes predictable | Same button same range | Vary timing and range each exchange |
| st.HK at unsafe spacing | Autopilot | Only use st.HK as a response to a whiff |
| Buffering DR from bad range | Greed | Confirm hit before committing DR |
| Wrong cancel timing | Rushed inputs | Drill cancel window isolation before full sequence |
