# Ryu Normals Guide

## Goal

Map each core normal to range, purpose, and risk.

## Core Normals (starter)

| Normal | Role | Primary Range | Notes |
|--------|------|---------------|-------|
| st.MP | pressure / confirm | close-mid | check walk-ins |
| cr.MK | poke / cancel route | mid | confirm into drive rush when stable |
| st.HK | control / counter poke | mid | punish whiffs, avoid autopilot |
| cr.HP | anti-air / challenge | close | spacing sensitive |

## Common Errors

- Repeating the same poke rhythm
- Using st.HK at unsafe spacing
- Buffering from bad ranges
