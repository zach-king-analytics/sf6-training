# Ryu Punish Combos

**Definition:** Combo routes executed when the opponent commits a punishable action (unsafe block string, whiffed move). Route selection depends on available Drive gauge, screen position, and spacing.

## Failure Modes

| Failure | Cause | Correction |
|---------|-------|------------|
| Going for optimal before consistent | Greed | Stick to one route per bucket until it lands reliably |
| Missed confirm on blocked unsafe | Not recognizing the window | Study punish frames for common unsafe moves |
| Wrong route for spacing | Autopilot | Note which route you attempted in loss entries |
| No punish at all | Hesitation | Even a single cr.MK counts — get something |
