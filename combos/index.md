## Notation Key

| Symbol | Meaning |
|--------|---------|
| `>` | Normal link (no cancel) |
| `~` | Cancel into next move |
| `xx` | Special cancel |
| `DR` | Drive Rush cancel |
| `OD` | Overdrive (EX) version |
| `DP` | Dragon Punch (623+P) |
| `Tatsu` | Tatsumaki (214+K) |
| `cr.` | Crouching |
| `st.` | Standing |
| `j.` | Jumping |
| `LP/MP/HP` | Light/Medium/Heavy Punch |
| `LK/MK/HK` | Light/Medium/Heavy Kick |
