# Drill: Punish Optimal

- drill_id: punish-optimal
- category: combo
- target_stage: Consistent

## Goal

Build one stable punish route per resource state.

## Setup

- Record dummy unsafe options
- Practice meterless, Drive Rush, and corner routes

## Reps

- 3 scenarios x 20 reps each

## Pass Criteria

- >= 85% successful punish conversion per scenario

## Failure Notes

- Wrong route selection
- Range misread
