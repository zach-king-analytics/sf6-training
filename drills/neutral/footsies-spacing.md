# Drill: Footsies Spacing

- drill_id: footsies-spacing
- category: neutral
- target_stage: Consistent

## Goal

Improve spacing discipline and whiff punish consistency.

## Setup

- Dummy: walk in/out + poke recording
- Focus on one primary whiff punish button

## Reps

- 5 sets x 20 neutral interactions

## Pass Criteria

- >= 70% correct spacing responses
- Reduced unsafe poke attempts

## Failure Notes

- Overextending into punish range
- Pressing at wrong rhythm
