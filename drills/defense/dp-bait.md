# Drill: DP Bait and Punish

- drill_id: dp-bait
- category: defense
- target_stage: Consistent

## Goal

Improve punish consistency after baiting reversals.

## Setup

- Dummy wake-up DP / block / throw tech mix
- Practice safe spacing and punish timing

## Reps

- 4 sets x 20 reps

## Pass Criteria

- >= 80% correct bait -> punish sequence

## Failure Notes

- Acting too early after bait
- Punish route drops
